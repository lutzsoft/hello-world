## Einige Git Kommandos: 
 - #### Erstmalig das Repository vom Server laden:
   ```git clone https://github.com/3rdof4/dusty.git```
 - #### Wechsel des Branch z.B. "debug", welcher auf dem Server bereits gibt
   ```git checkout --track origin/debug```
 - #### Wechsel des Branch zurück nach Master 
   ```git commit -m "<bermerkung optional>"```<br/>
   ```git checkout --track origin/master```
 - #### Anzeige wechsel des aktuellen branch
   ```git status```
 - #### Änderungen vom Server herunter laden
   ```git pull```
 - #### Änderungen local speichern
   ```git commit -m "<bermerkung optional>"```
 - #### Änderungen zum Server senden
   ```git push```
